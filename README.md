# Heart fialure eda learning
## Objective
The objective of this project is to practice explortory data analysis (EDA) on a clincal dataset related to heart failure, with a focus on data inspection , descriptive statstics, and basic visualization techniques.
## Dataset
This project uses a publicly available heart failure clincal dataset obtained from Kaggle. The dataset contains demographic variables , clincal measurements, and outcome-related features commonly used in heeart failure studies.
